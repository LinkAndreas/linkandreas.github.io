Welcome to my personal portfolio. This repository contains the source code for [linkandreas.de](https://www.linkandreas.de), built with Next.js and React. It serves as a hub for my mobile applications, technical articles, and professional publications.

## 🚀 Key Projects

### [ImgZen](https://www.linkandreas.de/imgzen)
**The Image Converter** — A native iOS/iPadOS application designed for effortless image format conversion.
- Support for PNG, JPEG, TIFF, BMP, HEIC, and WebP.
- Batch processing and privacy-focused local conversion.

### [Timity](https://www.linkandreas.de/timity)
**The missing Time Calculator** — An essential tool for calculating durations, event offsets, and solar times.
- Event list tracking and duration calculation.
- Integrated solar calculator for sunrise and sunset.

## 🏗 Setup & Development

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm

### Instructions

1. **Clone the repository**:
   ```sh
   git clone https://github.com/LinkAndreas/linkandreas.github.io.git
   cd linkandreas.github.io
   ```

2. **Install dependencies**:
   ```sh
   npm run bootstrap
   ```

3. **Run the development server**:
   ```sh
   npm run dev
   ```
   The site will be available at `http://localhost:3000`.

4. **Clean the workspace**:
   ```sh
   npm run clean
   ```
